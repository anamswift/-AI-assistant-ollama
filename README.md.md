## 🚀 How to Run (Every Time)

### Step 1 — Open your folder in VS Code
**File → Open Folder → streamlit interface** (on Desktop)

### Step 2 — Open terminal
Press **Ctrl + `**

### Step 3 — Set the path
```bash
cd "C:\Users\DELL\Desktop\streamlit interface"
```

### Step 4 — Run the app
```bash
streamlit run ollama_chat.py
```

Your browser opens automatically at **http://localhost:8501** 🎉
